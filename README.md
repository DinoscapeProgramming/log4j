# Log for JavaScript (log4j)
A simple log package for JavaScript

## Documentation
### Require the package
```js
const log = require('./log4j.js')
```

### Logging a simple message
```js
log("Hey Guys!")
```
#### Result
```
Hey Guys
```

### Using elements
```js
log("Hey {}! What's up {}", ["Guys", "lol"])
```
